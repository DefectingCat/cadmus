# helpers 目录

测试辅助工具目录，为项目单元测试提供通用辅助函数和测试数据 fixtures。

## 目录结构

```
helpers/
└── doc.go          # 包声明文件
```

## 文件说明

### doc.go

`doc.go` 声明 `helpers` 包，用于存放测试辅助函数和 fixtures。

```go
// Package helpers provides test utility functions and fixtures.
package helpers
```

## 用途

helpers 包用于存放：

- 通用测试辅助函数
- 测试数据生成工具
- 测试 fixtures 和 mock 数据
